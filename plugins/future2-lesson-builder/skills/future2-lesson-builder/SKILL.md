# Future 2 Lesson Builder

The operational rulebook for building an HISD Future 2 Experiences lesson plan
(grades 3-8) so it (a) stays true to HISD's fixed Scope & Sequence, (b) meets
the Access Model bar, and (c) ships in the locked, HISD-readable doc format.
Apply this to **every** lesson-plan write, rewrite, reconciliation, or review.

Governing philosophy lives in `future2-lesson-generation` (write canonical first,
render derivative, run the fidelity gate) and `lesson-injection` (clarity/quality
bar). This skill is the mechanical spec that sits on top of them.

## Authoritative source — the LOCKED HISD Scope & Sequence

**Every lesson written from here on must align to this one sheet:**

> **`17UFR-Lzd4Lupz6lgsJ_d7_bJcnbA_D23_ZV8tLCaCwo`** — "Cycle 1 Scope & Sequence"
> (tabs `34` / `56` / `78`, one per grade band; each holds both that band's A1 and B1).

It is the single source of truth for the **fixed columns** (Day, Lesson Title,
Lesson Design Type, LO, Rubric Indicator, Connection). Before writing or
reconciling any lesson, pull that day's row from this sheet and match the fixed
columns **exactly**; only EO / Activity / Materials / Est. Cost may reflect the
build. If any earlier draft or doc disagrees with this sheet on a fixed column,
this sheet wins (except the user-authorized staff-internalized titles in Rule 11).

## Rule 1 — Fixed vs. alterable columns (the core discipline)

The Scope & Sequence has 10 columns. Only four may ever be changed.

- **FIXED — honor HISD exactly, build around them (all map to the rubric):**
  Day, **Lesson Title**, **Lesson Design Type**, **LO** (Lesson Objective),
  **Rubric Indicator**, **Connection**.
- **ALTERABLE — the only thing you may change is *how students meet the LO*:**
  **EO** (Experience Objective), **Activity**, **Materials Needed**, **Est. Cost**.

Never edit a fixed column to fit your activity. Plan to the **LO** above all.
If a reconciliation shows a fixed column drifted from HISD, revert it to HISD's
value — do not "improve" it. Experiences must **escalate in challenge over time**.

Exception: **staff-internalized titles** may override HISD's title where the user
has explicitly authorized it (see Rule 11). Document every such exception.

---

## Rule 2 — Lesson Design Type must equal the S&S, and Skills Lab means scored

The doc's **Lesson Type** row MUST equal the S&S **Lesson Design Type** for that day.

| S&S Design Type | What the lesson is | Doc behavior |
| --- | --- | --- |
| Gradual Release with Discussion | Concept intro | Ends with an **embedded low-stakes practice bridge**, not a reflection prompt |
| Guided Action | A simulation/practice or performance sim | Teacher **coaches**, re-dos allowed, **not scored** |
| Skills Lab | An **assessed checkpoint** of the rubric indicator | **Scored** assessment, no coaching during the scored run |
| CPC — Live Performance | Multi-day culminating check | Highest stakes; **never shares the "Skills Lab" label** |

- **Skills Lab is reserved for checkpoints and is ALWAYS a scored assessment** of
  the rubric indicator. Never label a mid-block practice/sim day "Skills Lab."
- Never set a non-checkpoint day to Skills Lab, and never build a scored run on a
  Guided Action day — scoring belongs to the checkpoint (usually Day 8, 12, 16, 20).
- The locked repeating cycle per indicator: Gradual Release (+ embedded practice)
  → Simulation (Invisible) → Simulation (Practice/Performance) → **Checkpoint
  (Skills Lab)**, then the CPC on *All Indicators* at experience end.

## Rule 3 — Materials must be low-cost and scalable

Every lesson must run in a room of **20-30 students (5-7 groups)** on common,
low-cost, easily-acquired campus supplies.

- **Good:** short lists of paper, cardstock, cardboard, tape, string, cups,
  straws, index cards, rubber bands, markers, scissors, paper clips, washers.
- **Banned / re-vehicle:** specialty or one-per-group big items (a fan, foam pipe
  insulation, LED+battery kits, model figures, tents), lists of 10+ items per
  group, and **any food** (no marshmallows — use a light block/ball).
- **Realistic costs.** No $0 for tangrams, no $5 for LEGO. State a believable
  per-class or per-team figure, "reusable" where true.
- **Leave HISD-origin items as HISD wrote them** — Days 1-5, the CPC, and HISD's
  kept activities (e.g., pool noodles, LEGO, a lock box). Only re-vehicle our
  own authored materials.

---

## Rule 4 — Connection is fixed, present, and written forward

- Every lesson-plan doc's **EXPERIENCE OVERVIEW must include a Connection row.**
  (Docs that omit it are non-compliant.)
- The Connection text must be **HISD's fixed Connection value** for that day and
  match the S&S Connection cell exactly.
- Write it **forward**: name the day/checkpoint/CPC this lesson loads, and what a
  student who skips today would be missing later. A backward summary is not a
  connection. Name A/B same-week crossovers where they exist.

---

## Rule 5 — A real hands-on core with a testable stake

Every dyad needs a **real physical build/do core with a binary "win test,"** not
discussion + a plan. "The object works or it doesn't; the mission is complete or
it isn't." Discussion and reflection wrap the experiential core; they never
replace it. Runs at 30:1 with no bottleneck adult and no adult rescue that
removes the learning. Use AI **only** where the S&S marks it (Rule 7).

---

## Rule 6 — Per-indicator through-line moves (escalate by band)

Carry the indicator's named move across its whole unit; escalate the demand by
grade band.

**Teamwork & Collaboration (A1)**
- Perspective Integration: **Build-On Protocol — Name, Build, Bridge** (3-6);
  **Steelman, Overlap, Synthesize** (7-8, synthesize *competing* views into
  dual-purpose solutions).
- Shared Accountability: **Claim, Do, Prove** (own the outcome/seams, not just a task).
- Persistence & Adaptability: **Test, Name, Adjust**.
- Productive Conflict & Challenge Navigation: **State, Hear, Bridge**.

**Emotional Intelligence (B1)**
- Self-Management: **Name, Match, Move** (+ co-regulation **Notice, Offer, Redirect** at 7-8).
- Social Awareness: **Look, Name, Respond** (name the *need* under the behavior).
- Relationship Building: **Own, Repair, Reconnect**.
- Responsible AI & Digital Wellbeing: **Ask, Check, Decide**.

---

## Rule 7 — AI: 3-stage autonomy arc, placed by the S&S

Read AI placement straight from S&S **Column F markers** ("AI Integration #N
included"). Do not add AI where the S&S has no marker.

| Stage | Name | Student does | Guard |
| --- | --- | --- | --- |
| 1 | Witness | Evaluates an AI output against their own experience | Debrief the gap |
| 2 | Thought Partner | Pressure-tests a decision they already made | Watch for over-reliance |
| 3 | Auditor | Challenges AI, finds where it fails, defends own judgment | Step back |

**Design test:** a student must NOT be able to pass by doing whatever the AI says.
Physically test AI advice; keep/adapt/reject with evidence. State the stage, the
decision point, and the misuse signal for every AI integration.

---

## Rule 8 — Grade-band escalation

Mechanism stays the same; the demand hardens across 3-4 → 5-6 → 7-8 through
complexity, interdependence, responsibility, or independence. A single-band build
where escalation is expected is a dilution.

---

## Rule 9 — Binary, observable assessment

Success is proven against an external bar (a win test), not by completion.
Skills Labs and the CPC are **scored**; practice/sim days are **formative** and
feed the checkpoint. On a scored run the teacher scores and does not coach.

## Locked doc format

Build the doc from `assets/lesson-plan-template.html` (exact palette, fonts,
bands, and section order). Create it as a Google Doc via `create_file` with
`contentMimeType: "text/html"` and the HTML in **`textContent`** (see Tooling).

**EXPERIENCE OVERVIEW rows, in order:** Lesson Type · Competency · Rubric
Indicator · Lesson Objective (LO) · Experience Objective (EO) · **Connection** ·
Materials needed. Then **MATERIALS DETAIL**, **LESSON FLOW**, **DIFFERENTIATION**,
**EVIDENCE**.

**Default 55-minute flow** (Simulation/Skills Lab/CPC):
Do Now (~8) → Direct Teach (~7, teacher talk < 90s) → Guided Practice (~12) →
Independent Practice / earned unlock (~18) → Closing (~10). Every flow block
carries **time-stamped [MM:SS] teacher directions**, sentence stems, and stop
points. **Gradual Release** replaces the final reflection with the low-stakes
practice bridge.

**Voice & formatting:** Calibri; say **"Teacher"** not "Guide"; **no em dashes**;
**no horizontal rules**. Early finishers get an **earned unlock**, never filler.

## Rule 11 — Staff-internalized title exceptions

Preserve these staff-learned titles even over HISD's S&S title (user-authorized),
and keep the S&S title cell matching them:
- 5-6 A1 D7 = **Hidden Information Investigation**
- 5-6 A1 D8 = **Checkpoint 2: Stakeholder Decision Challenge**
- 5-6 B1 D6 = **Regulation Toolbox Lab**
- 7-8 A1 D6 = **Stakeholder Town Hall**
- 7-8 B1 D7 = **Leadership Under Stress**

Any new exception must be explicitly user-authorized and documented. Everywhere
else, the title is HISD's fixed value.

## Tooling constraints (Google Drive MCP)

- Create a doc: `create_file` with `contentMimeType: "text/html"`, HTML in
  **`textContent`** only. **Do not also pass `content`** (an empty `content` writes
  an empty doc). A freshly created Doc may report `fileSize: "1"` — that is a
  metadata lag, not an empty file; verify with `read_file_content` if unsure.
- There is **no cell-edit, no delete/trash, no rename, and no multi-tab sheet**
  API. Sheets can only be made from single-tab CSV via `create_file`. To fix S&S
  cells, hand the user an exact cell-address edit list; to remove a superseded
  doc, ask the user to delete it manually.
- **One canonical doc per day.** When you recreate a lesson, record the new ID
  and flag the superseded one(s) for deletion so staff never grab the wrong plan.
- Some docs carry a sensitivity label ("ineligible for generative AI contexts")
  and cannot be read/edited — recreate fresh instead.

## Pre-ship checklist (run before declaring a lesson done)

1. Doc **Lesson Type = S&S Design Type**; Skills Lab ⇒ scored, non-checkpoint ⇒ not Skills Lab.
2. **Fixed columns untouched** (Day, Title, Design Type, LO, Rubric Indicator, Connection); only EO/Activity/Materials/Cost reflect the build.
3. **Connection row present** and equal to HISD's Connection, written forward.
4. **Materials scalable** for 20-30 students, low-cost, no food, no specialty/10+ item lists; HISD-origin items untouched.
5. A **real hands-on core with a binary win test**; no adult rescue; runs at 30:1.
6. The indicator's **through-line move** is used and **escalated** for the band.
7. **AI only where the S&S marks it**, correct stage, and un-passable by obeying AI.
8. **EO/Activity/Materials/Cost match the delivered doc** (S&S and doc stay in sync).
9. Format: locked template, "Teacher," no em dashes, no horizontal rules, time-stamped flow, earned unlock.
