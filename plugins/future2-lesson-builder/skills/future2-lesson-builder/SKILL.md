# Future 2 Lesson Builder

The operational rulebook for building an HISD Future 2 Experiences lesson plan
(grades 3-8) so it (a) stays true to HISD's fixed Scope & Sequence, (b) matches
the assigned **lesson design type's anatomy exactly**, and (c) is detailed enough
that HISD's team can transfer it straight to slides with **no clarifying
questions**. Apply to every lesson write, rewrite, reconciliation, or review.

Governing philosophy lives in `future2-lesson-generation` (write canonical first,
render derivative, run the fidelity gate) and `lesson-injection` (clarity bar).

Two things HISD requires and this skill enforces:
1. **Content + structure match the S&S lesson design type exactly** (Rule 2).
2. **Enough detail to transfer to slides with zero follow-up** (Slide-deck format).

## Rule 1 — Fixed vs. alterable columns

Only four S&S columns may change.
- **FIXED (honor HISD exactly):** Day, **Lesson Title**, **Lesson Design Type**,
  **LO**, **Rubric Indicator**, **Connection**. Plan to the LO above all.
- **ALTERABLE (how students meet the LO):** **EO**, **Activity**, **Materials
  Needed**, **Est. Cost**.

Never edit a fixed column to fit an activity; if one drifted, revert it to HISD's
value. Experiences escalate in challenge over time. Staff-internalized title
exceptions only where user-authorized (Rule 11).

---

## Rule 2 — Design-type anatomy (structure + timing MUST match the S&S type)

The doc's Lesson Type = the S&S Lesson Design Type, and the doc's **block
structure and timing must follow that type's anatomy exactly**. HISD color names:
**Tangerine = Direct Teach · Purple = Guided · Green = Independent** (Do Now opens;
reflection + a closure slide close every type).

### Gradual Release with Discussion — *discussion of the skill; starts a new rubric indicator*
- Do Now **3-5** · Tangerine (Direct Teach) **10** · Purple (Guided) **15-20** · Green (Independent) **25-30** · Reflection · Closure.
- Introduce the new rubric indicator (covers the next few days); give teaching
  points for using the skill; **maximize student discussion** from the direct
  teach on. An activity in Green is **optional**, to deepen discussion.
- **Green discussion menu:** pair-share · pair-share with a new partner · pair
  then join another pair · Table Talk (fishbowl: one group talks, one listens).

### Guided Action — *hands-on application; practices a new rubric indicator in action*
- Do Now **3-5** · Tangerine **5-10** · Purple (Guided) **20-25** · Green (Independent) **20-25** · Reflection · Closure.
- Stamp key skill ideas in ~**3 slides** before the guided. Introduce the
  activity in Purple **with intentional teacher check-ins** that coach the skill.
  Release in Green to complete it, **chunked so the stakes rise multiple times**.

### Skills Lab — *hands-on assessment; scores the covered rubric indicator*
- Do Now **3-5** · Tangerine (Direct Teach) **5** · Green (Independent) **40-45** · Reflection · Closure.
- **NO Purple / guided section.** Direct teach reviews **how they'll be scored
  (the rubric indicator)** and explains the activity; then a **very hands-off**
  Green block, chunked so stakes rise. Mark **where scoring begins** on-screen and
  make the "scored on your behaviors/skill, not whether the build wins" point
  explicit. Teacher observes and scores; **no coaching** once the checkpoint starts.

**CPC — Live Performance** is a separate multi-day escalation, locked elsewhere
(the CPC designs); do not fold it into a single-day Skills Lab.

**Universal (every type):** an **LO slide** (Competency + Rubric Indicator +
Lesson Design Type; LO = what am I learning, rubric-aligned; EO = what am I doing,
action/task) · a **Materials slide** · **sentence stems on the slide** · a
**student-engagement-strategy indicator** on each slide · a clear split of what
the student does vs. what the teacher does on every slide.

## Rule 3 — Materials must be low-cost and scalable

Runs for **20-30 students (5-7 groups)** on common campus supplies (paper,
cardstock, cardboard, tape, string, cups, straws, index cards, rubber bands,
markers, scissors, paper clips, washers). **Banned:** specialty/one-per-group big
items (fan, foam pipe insulation, LED/battery kits, model figures, tents), lists
of 10+ items per group, and **any food** (no marshmallows). Realistic costs.
**Leave HISD-origin items as HISD wrote them** (Days 1-5, CPC, kept activities).

---

## Rule 4 — Connection is fixed, present, and written forward

Carry HISD's fixed **Connection** into the planning header of every lesson doc,
matching the S&S cell exactly, written **forward** (name the day/checkpoint/CPC it
loads and what a skipper would miss). Name A/B same-week crossovers.

---

## Rule 5 — A real hands-on core with a testable stake

A real physical build/do core with a **binary win test**, not discussion + a plan
(Gradual Release excepted, which is discussion-centric by design). Runs at 30:1,
no bottleneck adult, no rescue that removes the learning.

---

## Rule 6 — Per-indicator through-line moves (escalate by band)

Carry the indicator's named move across its whole unit; the checkpoint scores
**that** move, not generic behavior.

**A1:** Perspective Integration — **Name, Build, Bridge** (3-6) / **Steelman,
Overlap, Synthesize** (7-8). Shared Accountability — **Claim, Do, Prove**.
Persistence & Adaptability — **Test, Name, Adjust**. Productive Conflict —
**State, Hear, Bridge**.
**B1:** Self-Management — **Name, Match, Move** (+ **Notice, Offer, Redirect**
co-regulation at 7-8). Social Awareness — **Look, Name, Respond** (name the need).
Relationship Building — **Own, Repair, Reconnect**. Responsible AI — **Ask, Check,
Decide**.

---

## Rule 7 — AI: 3-stage arc, placed by the S&S

Read AI placement from S&S **Column F markers** ("AI Integration #N included").
Add AI only where marked. Stage 1 Witness (evaluate an AI output) → Stage 2
Thought Partner (pressure-test a decision already made) → Stage 3 Auditor
(challenge AI, defend own judgment). Physically test AI advice; keep/adapt/reject
with evidence. **A student must not be able to pass by obeying the AI.** State the
stage, the decision point, and the misuse signal.

---

## Rule 8 — Grade-band escalation

Same mechanism, harder demand across 3-4 → 5-6 → 7-8 (complexity, interdependence,
responsibility, or independence). Single-band builds are a dilution.

---

## Rule 9 — Binary, observable assessment

Success proven against an external bar (a win test), not completion. Skills Labs
and the CPC are **scored**; Gradual Release and Guided Action days are
**formative** and feed the checkpoint. On a scored run the teacher scores and does
not coach.

## Locked doc format — the slide-deck lesson plan

Build from `assets/lesson-plan-template.html`. The doc is a **slide-by-slide build
sheet** HISD copies straight onto slides. Publish as a Google Doc via `create_file`
(`contentMimeType: "text/html"`, HTML in **`textContent`** only — see Tooling).

**1. Planning header (top of doc, not a slide)** — our fidelity layer:
Competency · Rubric Indicator · Lesson Design Type · LO · EO · **Connection** ·
Materials summary · Lesson length. (Design Type here drives the block structure/
timing per Rule 2.)

**2. Slide-by-slide body**, grouped into the design type's blocks (Do Now ·
Tangerine/Direct Teach · Purple/Guided [omit for Skills Lab] · Green/Independent ·
Reflection · Closure). **Every slide carries these fields:**
- **Slide N — Title**
- **Purpose** (one line)
- **Time**
- **On Screen** — the *verbatim* slide copy (headline + bullets), ready to paste
- **Visual** — the illustration/diagram/icons to place
- **Student Engagement Strategy** — what students do (the icon/strategy) + **Sentence Stem/Starter** (goes on the slide)
- **Teacher Guidance** — what the teacher does
- **Teacher Script** — the exact words to say
- **Possible Student Responses** (and Questions/Actions where useful)
- **Evidence to Observe** — for Skills Labs: what is scored and *when scoring starts*
- **Transition** — the exact words that move to the next slide

Include an early **LO slide** (Competency + Rubric Indicator + Design Type + LO +
EO on screen) and a **Materials slide**. Per-slide times sum to each block's
budget; block budgets follow the Rule 2 anatomy for the type.

**Voice & formatting:** Calibri; say **"Teacher"** not "Guide"; **no em dashes**;
**no horizontal rules**; early finishers get an **earned unlock**, not filler.

Preserve these over HISD's S&S title (user-authorized); keep the S&S title cell

New exceptions must be user-authorized and documented.

## Tooling constraints (Google Drive MCP)

- Create a doc: `create_file`, `contentMimeType: "text/html"`, HTML in
  **`textContent`** only. **Do not also pass `content`** (empty `content` writes an
  empty doc). A new Doc may report `fileSize: "1"` — metadata lag, not empty;
  verify with `read_file_content`.
- No cell-edit, no delete/trash, no rename, no multi-tab sheet API. Sheets only
  from single-tab CSV. To fix S&S cells, hand the user an exact cell-address list;
  to remove a superseded doc, ask the user to delete it.
- **One canonical doc per day.** On recreate, record the new ID and flag the
  superseded one(s) for deletion.
- Some docs carry a sensitivity label ("ineligible for generative AI contexts")
  and cannot be read/edited — recreate fresh.

## Pre-ship checklist

1. Doc Lesson Type = S&S Design Type, and **block structure + timing match that
   type's anatomy** (Rule 2). **Skills Lab has no guided section** and marks where
   scoring begins.
2. **Fixed columns untouched**; only EO/Activity/Materials/Cost reflect the build.
3. **Planning header present** with **Connection** = HISD's, written forward.
4. **Every slide has On Screen + Visual + Teacher Script + Possible Student
   Responses + a sentence stem + engagement strategy** — transfers to slides with
   no clarifying questions.
5. LO slide (Competency + Indicator + Design Type + LO + EO) and Materials slide present.
6. **Materials scalable** (20-30 students, no food/specialty/10+ lists); HISD-origin untouched.
7. Real hands-on core + binary win test (Gradual Release = discussion); 30:1, no rescue.
8. The indicator's **through-line move** is used and **escalated**; the checkpoint scores that move.
9. **AI only where the S&S marks it**, right stage, un-passable by obeying AI.
10. EO/Activity/Materials/Cost match the delivered doc (S&S and doc stay in sync).
11. Format: "Teacher," no em dashes, no horizontal rules, earned unlock.
